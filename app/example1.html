    • Features

    • Mobile ready
    • Hardware accelerated animation for silky smooth movements
    • Easily customizable look & feel - CSS easy to override
    • Small size - 6k (2k gZipped)
    • Few dependencies - only jQuery
    • AJAX tab loading on load or lazy loading
    • Fully accessible - keyboard navigatable with proper elements
    • Fully customizable tabLink types via data-attributes (see examples)
      • default link (drawer opens to this tab at first)
      • mobile only link
      • desktop only link
      • toggle drawer link
      • open drawer link
      • close drawer link
      • disabled link
      • 2 links opening 1 tab support
    • Fully configurable drawer behavior
      • jQuery fallback animation speeds (CSS3 transitions configurable via CSS)
      • Mobile breakpoint control for when the drawer goes full height
      • Other minor settings
    • FOUC Free (no flashes of unstyled content - good for slower browsers)

    Upcoming Features (listed by priority)

    • Auto Height (currently only does fixed height drawers)
    • (Possible)Greensock animation plugin for better mobile scrolling performance
    • (Possible) Manual resizing of fixed height drawers plugin with drag and drop
    • (Possible) tab transition plugins
